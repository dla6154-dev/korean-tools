## 추천 구현 순서

1. **글자수 세기** — 순수 프론트엔드, 수요 높음, 구현 빠름
2. **BMI/칼로리 계산기** — 현재 패턴 그대로 확장
3. **디데이 카운터** — 기념일 계산기와 시너지
4. **이미지 압축** — 클라이언트 사이드 구현 가능 (Canvas API)
5. **파일 변환** — 유료 플랜 전환 후 추가
